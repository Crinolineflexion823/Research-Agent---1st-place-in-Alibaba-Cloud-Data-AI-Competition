## 🔎 Main features

### 🔁 Multi-hop reasoning
The app can handle questions that need more than one step. It can gather clues, then use them together to reach an answer.

### 🌐 Dual-engine search
It searches with two sources or search paths, which helps find more complete results and reduce gaps.

### 📄 Web content extraction
It can open web pages and pull the main text from them. This helps skip menus, ads, and clutter.

### 🧭 ReAct-style flow
The app can think through a task, act on it, then use what it finds to decide the next step. This helps with hard questions that need careful research.

### 🤖 Qwen model support
The project uses Qwen models for language tasks, which helps the app read, reason, and write answers.
